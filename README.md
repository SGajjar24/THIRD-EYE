A comprehensive security and content auditing system designed to reveal unseen vulnerabilities and detect synthetic content origin with high precision.

## Project Overview

THIRD EYE provides automated security risk assessments and deep-dive analysis into content authenticity. By auditing target URLs, the system identifies critical vulnerabilities and evaluates the probability of human vs. synthetic (AI) content generation.

For a detailed technical breakdown, visit our [Kaggle Project Writeup](https://www.kaggle.com/competitions/gemini-3/writeups/THIRD-EYE).

## Features

- **Security Risk Assessment**: Automated vulnerability posture scoring and critical risk detection.
- **Content Origin Analysis**: Advanced detection of AI-generated content (Synthetic Detection).
- **Comprehensive Reporting**: Generates interactive HTML reports for detailed auditing.
- **Multi-Media Insights**: Includes video demonstrations and PDF documentation for ease of understanding.
